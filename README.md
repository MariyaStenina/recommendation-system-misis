# recommendation-system-misis
## Описание

Задача - разработать рекомендательную систему для процесса согласования
документов в «Первая Форма»
Рекомендательная система в BPM – платформе позволяет повысить скорость при принятии решений для различных бизнес – процессов, а также положительно влияет на прибыль организации, осуществляя рекомендацию клиентам по товару или услуге. В приложении «Первая форма» инструменты интеллектуального анализа не реализованы, поэтому разработки в этой области представляют большой интерес для пользователей системы.
Основная идея BPM – систем состоит в том, чтобы поддерживать взаимосвязь таких элементов, как контроль, планирование и проектирование. Система разделяет работы на функциональные блоки, устанавливает исполнителей и время на выполнение, обеспечивая, таким образом, управление бизнес – процессом. Такой принцип построения систем называется модульным. Модульное построение предполагает наличие средств визуализации, инструментов мониторинга, систем управления и возможностей для совместной работы в системе.
«Первая Форма» включает в себя целый набор инструментов, с помощью которых можно автоматизировать различные процессы. Например, можно создавать приложения для гибкого управления проектами, реализовать систему электронного документооборота, корпоративный портал, а также связать множество смежных процессов.



## Результат

В результате работы была разработана рекомендательная система для
BPM – платформы «Первая форма», внедрение которой позволит организации определять для клиентов наиболее актуальные услуги и прогнозировать спрос на них. Понимание того, какие услуги наиболее востребованы, приведет к их эффективной продаже, увеличивая прибыль компании.
